**xxxxxxxx*! = cambiar el numero del usuario
*#xxxxxxxx*! = cambiar el numero del cliente
*@xxxxxxxx*! = cambiar el numero del usuario2
*?xxx*! = Cambiar la medida del sensor minima
*$xxx*! = Cambiar la medida del sensor maxima
*Txxxxx*! = Cambiar el intervalo de tiempo en minutos seguridad en caso de no recibir el mensaje o perdida
*txxxxx*! = Cambiar el intervalo de tiempo en minutos seguridad en caso de no recibir el mensaje o perdida
*Level?  = muestra el nivel del tanque


*%? = mostrar la configuracion del sistema

!!!ON = cliente sensar
!!#OFF = Dejar de sensar


@@1*! = cliente pide carga de agua
@#0*! = cliente pide dejar de dispensar agua


&&&ON_BOMBA*! = Server avisa que la bomba se encendio mediante un msn de texto
&&&OFF_BOMBA*! = Server avisa que la bomba se encendio mediante un msn de texto

*sta=void = Función exclusiva para hacer mantenimiento, permite establecer el inicio de la medida si el tanque esta vacio
*sta=full = Función exclusiva para hacer mantenimiento, permite establecer el inicio de la medida si el tanque esta lleno
